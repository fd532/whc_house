# 武汉买房攻略

## 背景
之前在武汉买房的时候，有过不好的体验，找房其实蛮不好找的。市面上有很多平台，比较火的就是链家与贝壳的，里面关于楼盘的介绍大多数都是优点，可信度还是蛮低的。很多的信息都是从一些社区或者跟朋友沟通才能了解。

所以想开发一个网站，用于介绍武汉楼盘的信息，不仅有排名，有位置，有商家的介绍，同时希望能存在社区的讨论，国家政策信息等等（这很难）。类似于一些平台的景评项目，希望能通过评论能给大家提供真实的楼盘信息。

## 前端展示
Electron + Koa + Vue(+🐔)
